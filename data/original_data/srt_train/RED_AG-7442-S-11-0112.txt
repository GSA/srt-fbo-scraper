Terrestrial Laser Scanner System – LIDAR

Post Processing / Analysis Software

	The USDA-NRCS Texas State Officer has a requirement for a 3D Ground based laser Scanner / Imaging System (LIDAR) post processing software suite, which shall be a LIDAR production manager for a Microsoft Windows system with a node locked license key.  The software suite shall be a non-standard set of tools to perform data calibration, X-Y-Z data matching, automatic filtering, data re-projection, classification, asset management and project management tailored to address specific needs of USDA-NRCS. These needs and other potential special custom features designated by USDA-NRCS shall be incorporated or developed. These custom features and processes would include but is not limited to requirements stated.

	In order to be determined an acceptable proposal, they must meet the following minimum requirements:

Tier 1 Priority items related to LIDAR Data Calibration, Matching, Data Processing, Feature Extraction, Classification, Visualization & Application Specific Features for USDA_NRCS TX.

Standalone software to operate on Microsoft Windows based PC.

Supports airborne, mobile & terrestrial static LIDAR data.

Interactive GUI

Automatic control point enforces normalization and matching.

Detailed survey control points comparison and accuracy reports.

Option to choose Area of Interest

Automatic loading of AOI chosen

Filtering – Automatic, Intelligent and contextual filtering with the following

Reclassification of Points

Overlap Filter for Airborne Data

Filter ot Extract Bare Earth from Mobile LIDAR

Filter to classify by color (RGB)

Filter to classify by intensity

Option to obtain LAS output

Smart classification techniques to extract key features: tailored for USDA-NRCS TX

Batch filter operation using pre-recorded macros for different terrain types.

Option to filter by a set of tiles.

Option to filter by user defined polygon

Built-in Memory optimization

Visualization of very large datasets at one time

Fully customizable visualizing preferences.

One Button Visualization/feature extraction

Elevation (Z) bare earth

Intensity

Color (RGB)

Classification

Customized workflow tailored for USDA-NRCS TX special requirements

32 hours classroom style training
