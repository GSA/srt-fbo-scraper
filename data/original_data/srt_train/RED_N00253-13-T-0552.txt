MATERIAL SPECIFICATIONS Specifications for CSIPL Software (C-language Signal and Image Processing Library) 1. The required software package must have consist of libraries of assembly optimized digital signal processing (DSP) functions or modules for use with Intel i7 series processors, including the i7-2715QE and i7-3617UE.

N00253-13-T-0552 Page 4 of 44 These functions/modules must be callable from C-language programs. The functions/modules must be written to take maximum advantage of the Intel i7's 256-bit advanced vector accelerator (AVX). 2. Speed of Execution Requirement: The vendor must be able to demonstrate execution of a 1024 point Complex floating point FFT, in-place, in 1.82 microseconds, or less, using an Intel i7 processor with AVX, operating at 2 GHz.. These libraries will be used to replace similar libraries, supplied with most VME based DSP boards, that are specific to the same board vendor. More specifically these libraries must compile to run under VxWorks O.S. version 6.9 and under Linux O.S. (Debian version 7.0). (Note: We're talking about two libraries here, two CSIPLs, one for each O.S., Operating System, to be supported.) The following functions/modules must be provided as part of each library: (a) Vector move, integer: Moves contents of one integer vector to another integer vector. Striding parameters are required on the input and output, as well as the number of values to move. (b) Convert short integer to float: Converts 16-bit integer vector to 32-bit floating point vector. Number of values is a required parameter. (c) Convert integer to float: Converts 32-bit integer vector to 32-bit floating point vector. Number of values is a required parameter. (d) Vector move, float: Moves contents of one floating point vector to another floating point vector. Striding parameters are required on the input and output, as well as the number of values to move. (e) Vector add, float: Computes the sum, by element, of one floating point vector and another floating point vector. Striding parameters are required on both inputs and the output, as well as the number of values to add. (f) Vector multiply, float: Computes the product, by element, of a floating point vector and another floating point vector. Striding parameters are required on both inputs and the output, as well as the number of values to multiply. (g) Vector Maximum value with index, float: Finds the maximum value in a floating point vector and its associated index. Stride and number of values are required parameters. (h) Vector fill, float: Sets the elements of a floating point vector to a scalar value. Stride and number of values are required parameters. (i) Vector mean value, float: Returns the mean value of a floating point vector. Stride and number of values are required parameters. (j) Square root, float: Computes the square root of a scalar. (k) Sine, float: Computes sine of a scalar angle in radians. (l) Cosine, float: Computes cosine of a scalar angle in radians. (m) Complex vector move: Moves contents of one complex floating point vector to another complex floating point vector. Striding parameters are required on the input and output, as well as the number of values to move. (n) Complex vector add: Computes the sum, by element, of one complex floating point vector and another complex floating point vector. Striding parameters are required on both inputs and the output, as well as the number of values to add. (o) Complex vector multiply: Computes the product, by element, of a complex floating point vector and another complex floating point vector. Striding parameters are required on both inputs and the output, as well as the number of values to multiply. (p) Complex vector fill: Sets the elements of a complex floating point vector to scalar values. Stride and number of values are required parameters. (q) Complex vector magnitude: Computes the magnitude for each complex element of a complex floating point vector. Stride and number of values are required parameters. (Note: Same as complex vector absolute value.) (r) Complex vector and scalar multiply: Computes the product, by element, of a complex floating point vector and a scalar. Striding parameters are required for the vector input and the output, as well as the number of values to multiply.
